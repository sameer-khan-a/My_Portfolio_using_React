## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/username/my-portfolio-using-react.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-porfolio-using-react
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
